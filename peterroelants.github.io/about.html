About this blog

This blog started from some IPython notebooks with notes I made for myself. Starting from these notebook files it was only a small effort to publish them on Github Pages so that they might be of benefit to others.

This blog is made using Jekyll and is hosted on Github pages. To publish the IPython notebook files I created a small script that uses nbconvert to convert the notebooks into HTML.
